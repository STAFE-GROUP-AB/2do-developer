💡 **How Image Pasting Works:**
1. Copy an image to your clipboard
2. Start AI Redirector with: `airedirector start`
3. Choose 'chat' mode
4. Type 'image' to load an image file
5. Or if you have an image in clipboard, it will be detected automatically!

✨ **Features Demonstrated:**
• 📊 Rich image information display
• 🎨 ASCII art preview
• 💾 Automatic temporary file storage
• 🔗 Integration with chat and todo systems
